# ![StudioDP](https://img.shields.io/badge/StudioDP-Landing%20Page%20Premium-6A0DAD?style=for-the-badge&logo=github&logoColor=white)

Este repositório apresenta o projeto **Studio Dayane de Paula**, uma landing page premium desenvolvida para um estúdio de beleza, com foco em estética, sofisticação e conversão de clientes.

O projeto foi construído com o objetivo de oferecer uma experiência visual elegante, destacando serviços, portfólio e facilitando o contato direto via WhatsApp.

---

## ![Conteúdo](https://img.shields.io/badge/O%20que%20tem%20aqui-Estrutura%20do%20Projeto-000000?style=for-the-badge)

- Landing page completa e responsiva  
- Estrutura organizada em seções estratégicas (Hero, Sobre, Serviços, Galeria, FAQ e CTA)  
- Apresentação institucional do estúdio  
- Exibição de serviços com descrição e preços  
- Galeria interativa estilo carrossel contínuo  
- Seção de perguntas frequentes (FAQ) interativa  
- Botões de ação focados em conversão (WhatsApp e Instagram)  

---

## ![Funcionalidades](https://img.shields.io/badge/Funcionalidades-Interatividade%20e%20UX-6A0DAD?style=for-the-badge)

- Animações suaves com **Intersection Observer** (efeito reveal)  
- Navbar dinâmica com mudança ao scroll  
- Scroll suave entre seções  
- Destaque automático da seção ativa no menu  
- FAQ com comportamento de acordeão  
- Galeria com animação horizontal contínua  
- Botão flutuante de WhatsApp  
- Design focado em experiência do usuário (UX) e conversão  

---

## ![Design](https://img.shields.io/badge/Design-UI%20Premium-000000?style=for-the-badge)

- Interface moderna com estética **minimalista e sofisticada**  
- Paleta de cores voltada ao segmento de beleza:
  - Preto profundo (fundo)
  - Rosa suave (destaques)
  - Dourado (elementos premium)
- Tipografia elegante com:
  - Cinzel Decorative (títulos)
  - Playfair Display (subtítulos)
  - Inter (textos)
- Uso de variáveis CSS para padronização visual (design system)
